Project Summary – Mobile Sales Power BI Dashboard
This Power BI project presents a comprehensive analysis of Mobile Sales Performance, featuring three major report pages: Dashboard, MTD Report, and Same Period Last Year.
Each page provides interactive insights for business stakeholders to track sales, customer behavior, and product performance across various dimensions.

1. Dashboard – Overall Sales Performance
The main dashboard provides a high‑level overview of the total sales performance across different regions, models, payment methods, and time periods.

Key Insights
Total Sales: 769M
Total Quantity Sold: 19K
Transactions: 4K
Average Price: 40.11K

Highlights
Total Sales by City (Map visualization) showing top-performing regions like Chennai, Kolkata, Mumbai, etc.
Total Quantity by Month, displaying seasonal trends.
Ratings by Status, helping assess customer satisfaction.
Transactions by Payment Method, comparing UPI, Debit Card, Credit Card, and Cash.
Total Sales by Mobile Model, identifying best-selling models such as iPhone SE, OnePlus Nord, Galaxy Note 20.
Total Sales by Day, showing weekday patterns.

2. MTD Report – Month‑to‑Date Analysis
This page focuses on MTD (Month-to-Date) performance for a selected year, quarter, month, or day.

Key Metrics
MTD Sales: 23M
MTD Quantity: 588
Transactions: 110
Average Price: 39.95K

Highlights
A detailed line chart visualizing MTD Sales growth from Day 1 to Day 28 of the month.
Monthly, yearly, and quarter filters for dynamic analysis.
Helps users measure current month performance vs. expectations.

3. Same Period Last Year – Yearly & Monthly Comparison
This page highlights year-over-year performance using advanced DAX time intelligence.

Includes-
Total Sales vs. Same Period Last Year (Yearly Bar Chart)
Quarterly Comparison (Total Sales vs. SPLY)
Monthly Comparison for granular trend analysis
A supporting summary table showing:
Year
Quarter
Total Sales
Same Period Last Year Value

Purpose-
Identify YoY growth
Track seasonal changes
Compare current performance to last year on the same timeline


Overall Dashboard Value:-
This Power BI solution helps organizations:

Monitor sales KPIs in real time
Compare monthly, quarterly, and yearly performance
Track model-wise, region-wise, and payment-wise trends
Evaluate customer satisfaction using rating data
Analyze time-series trends using MTD and SPLY logic
